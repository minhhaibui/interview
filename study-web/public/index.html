🧠 Luyện tư duy lập trình

          
Giải các bài phỏng vấn kinh điển bằng JavaScript — viết hàm rồi chạy test thật ngay trong trình duyệt. Bí thì xem gợi ý nhiều cấp; bí nữa thì xem lời giải có giải thích.

          

          

        

        

      

    


    
    

      

        
🧩 Luyện IQ & Logic

        
Câu hỏi trắc nghiệm rèn tư duy: dãy số, quy luật, suy luận logic, toán nhanh. Chọn đáp án để biết đúng/sai kèm giải thích. Làm hết một lượt sẽ có điểm IQ ước lượng.

        

      

    


    
    

      

        
🔍 Đoán Output

        
Đọc snippet JavaScript rồi đoán kết quả in ra console. Rèn các điểm hay bị hỏi: event loop, closure, hoisting, async/await, type coercion, this. Chọn đáp án để biết đúng/sai kèm giải thích.

        

      

    


    
    

      

        
📡 Quiz API & HTTP

        
Trắc nghiệm kiến thức HTTP/REST hay bị hỏi khi phỏng vấn Backend: status codes, idempotency & safe methods, caching/ETag, auth, CORS, rate limit, pagination. Chọn đáp án để biết đúng/sai kèm giải thích.

        

      

    


    
    

      

        

          
🐛 Tìm & Sửa Bug

          
Đoạn code có một lỗi tinh vi — đọc, tìm bug rồi sửa và chạy test thật để kiểm. Rèn kỹ năng debug hay gặp khi phỏng vấn: off-by-one, ép kiểu, thiếu return, base case đệ quy…

          

        

        

      

    


    
    

      

        
🗄️ SQL Drill

        
Trắc nghiệm SQL hay bị hỏi khi phỏng vấn Backend: JOIN & anti-join, NULL, GROUP BY/HAVING, index & composite index, transaction/ACID/isolation level, normalization, window function, đoán kết quả query, EXPLAIN, deadlock, N+1. Chọn đáp án để biết đúng/sai kèm giải thích.

        

      

    


    
    

      

        
🖥️ CLI Quiz

        
Trắc nghiệm câu lệnh hay dùng & hay bị hỏi khi làm Backend: git (reset/revert/rebase/stash), docker (run/exec/logs/ps), kubectl (describe/rollout/apply), redis-cli (SCAN vs KEYS, SET EX, INCR), psql (\dt, EXPLAIN ANALYZE), bash (grep -r, tail -f, kill -9, chmod). Chọn đáp án để biết đúng/sai kèm giải thích.

        

      

    


    
    

      

        
🔁 Ôn câu sai

        
Mọi câu trắc nghiệm bạn từng chọn sai (Đoán output · API/HTTP · SQL · CLI) được gom về đây thành một phiên ôn tập trung. Trả lời đúng ở lần ôn này thì câu đó tự rời khỏi hàng đợi; còn sai thì giữ lại để ôn tiếp lần sau — đúng tinh thần học từ lỗi.
